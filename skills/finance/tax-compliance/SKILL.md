# 税务合规技能

## 技能概述

本技能旨在为非税务专业的律师和商务人员提供第一道税务风险防线。在审查合同或分析交易架构时，自动识别潜在的税务触发点、合规风险及税负成本。

## 核心能力

### 1. 合同涉税条款审查
- **发票条款**: 审查发票类型（专票/普票）、税率约定、开票时间与付款时间的衔接。
- **包税条款**: 识别"包税"、"税后价"等条款的法律效力与计算陷阱。
- **印花税**: 提示合同性质对应的印花税目及税率。

### 2. 交易模式税务分析
- **股权转让**: 提示个人所得税（20%）或企业所得税申报义务。
- **劳务 vs 服务**: 区分劳务报酬（个税预扣）与经营所得（个体户/公司），提示合规风险。
- **跨境交易**: 提示预提所得税、增值税及附加。

## 审查规则 (Rule-Based)

### 发票合规规则
```
如果 合同金额 > 0 且 未提及发票类型:
    标记为: 🟡 中风险
    提示: "建议明确约定发票类型（增值税专用发票/普通发票）及税率，避免抵扣争议。"
```

### 包税条款规则
```
如果 出现 "甲方承担所有税费" 或 "税后金额":
    标记为: 🔴 高风险
    提示: "包税条款可能导致税基倒算，实际成本大幅增加。建议测算税前金额并明确列示。"
    依据: 此时应按 (税后金额 / (1-税率)) 计算合同总额计算印花税等。
```

### 劳务外包风险
```
如果 个人签约提供长期服务 且 未注册个体户:
    标记为: 🟡 中风险
    提示: "存在被认定为事实劳动关系的风险，且涉及个人所得税代扣代缴义务。"
```

## 声明
本分析基于通用税法逻辑，不构成税务鉴证报告。复杂税务筹划请咨询注册税务师(CTA)。
