# 知识产权侵权分析指南

## 1. 著作权侵权判定

### 1.1 核心原则：接触 + 实质性相似 (Access + Substantial Similarity)
- **接触**: 被告是否有机会接触到原告的作品？(如已公开发表、曾有合作关系等)
- **实质性相似**: 
  - **思想与表达二分法**: 著作权法保护表达，不保护思想。
  - **比对内容**: 独创性的表达部分是否相似。

### 1.2 合理使用抗辩 (著作权法 第24条)
- 个人学习、研究或者欣赏；
- 适当引用（注明出处，不影响原作品正常使用）；
- 新闻报道；
- 课堂教学等。

## 2. 商标侵权判定

### 2.1 核心原则：混淆可能性 (Likelihood of Confusion)
- **商标近似**: 视觉、读音、含义上的相似度。
- **商品/服务类似**: 核心类别与关联类别。
- **显著性与知名度**: 知名度越高的商标，保护范围通常越宽。

### 2.2 常见侵权行为 (商标法 第57条)
- 未经许可在同一种商品上使用相同的商标；
- 未经许可在同一种商品上使用近似的商标，或者在类似商品上使用相同或者近似的商标，容易导致混淆的；
- 销售侵犯注册商标专用权的商品的；
- 伪造、擅自制造他人注册商标标识或者销售伪造、擅自制造的注册商标标识的。

## 3. 赔偿计算

### 3.1 计算顺序
1. **实际损失**: 权利人因被侵权所受到的实际损失。
2. **侵权获利**: 实际损失难以确定的，可以按照侵权人因侵权所获得的利益确定。
3. **许可费倍数**: 参照该知识产权许可使用费的倍数合理确定。
4. **法定赔偿**: 权利人的实际损失、侵权人的违法所得、权利使用费难以计算的，由人民法院根据侵权行为的情节判决给予五百万元以下的赔偿。

## 4. 证据收集清单
- 权属证明（登记证书、创作底稿）。
- 侵权公证/时间戳（网页截图、购买记录）。
- 知名度证据（广告宣传、获奖记录、销售合同）。
- 损失/获利证据（财报、销售数据）。
