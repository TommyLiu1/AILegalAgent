# 法律研究技能

## 技能概述

本技能用于进行专业的法律研究，包括法律法规检索、司法判例分析、法律解释研究，为法律问题提供权威的法律依据支持。

## 研究能力

### 1. 法规检索
- 宪法及宪法性法律
- 法律（全国人大及其常委会制定）
- 行政法规（国务院制定）
- 部门规章（国务院各部委制定）
- 地方性法规
- 司法解释
- 规范性文件

### 2. 判例分析
- 最高人民法院指导性案例
- 最高人民法院公报案例
- 各级法院典型案例
- 仲裁裁决案例

### 3. 法律解释
- 立法解释
- 司法解释
- 行政解释
- 学理解释

## 研究流程

### Step 1: 问题定性
```
分析法律问题的性质：
- 确定法律关系类型
- 识别涉及的法律部门
- 明确争议焦点
```

### Step 2: 法规检索
```
检索相关法律法规：
- 直接规定（明确适用的法条）
- 间接规定（可参照的法条）
- 原则规定（指导性条款）
```

### Step 3: 判例研究
```
检索相关判例：
- 相同类型案件
- 相似事实案件
- 最新裁判趋势
```

### Step 4: 综合分析
```
整合法规和判例：
- 法律适用分析
- 裁判观点归纳
- 结论与建议
```

## 使用示例

### 示例1：合同违约责任研究
```
用户: 供应商延迟交货两个月，我们可以主张什么权利？

AI: 我将从以下几个方面进行法律研究：

1. **法律依据**
   - 《民法典》第577条：违约责任的一般规定
   - 《民法典》第583条：继续履行的请求权
   - 《民法典》第584条：损害赔偿范围
   - 《民法典》第585条：违约金调整

2. **可主张的权利**
   - 要求继续履行
   - 主张违约金（如约定）
   - 主张实际损失赔偿
   - 解除合同（如符合条件）

3. **相关判例**
   [分析类似案例的裁判结果]
```

### 示例2：劳动纠纷法律研究
```
用户: 公司未签劳动合同，员工可以要求什么赔偿？

AI: 根据法律规定：

1. **法律依据**
   - 《劳动合同法》第82条：未签订书面劳动合同的双倍工资
   - 《劳动合同法》第14条：视为订立无固定期限劳动合同

2. **赔偿标准**
   - 自用工之日起超过一个月不满一年：每月支付二倍工资
   - 满一年不订立：视为订立无固定期限劳动合同

3. **时效注意**
   - 仲裁时效：1年（《劳动争议调解仲裁法》第27条）
```

## 注意事项

1. 法律法规可能存在修订，请注意法律的时间效力
2. 司法判例仅供参考，不具有法律强制力（指导性案例除外）
3. 具体案件需结合实际情况分析
4. 重大法律问题建议咨询专业律师
