# 尽职调查技能

## 技能概述

本技能用于对目标企业进行全面的尽职调查，收集分析企业工商、财务、法律、经营等信息，评估潜在风险，并生成专业的尽职调查报告。

## 适用场景

- 投资并购前的目标公司调查
- 重大交易对手方背景核查
- 供应商/合作伙伴资质审查
- 上市公司信息披露核查
- 诉讼对手方资产线索调查

## 调查维度

### 1. 工商信息
| 调查项 | 内容 |
|-------|------|
| 基本信息 | 企业名称、统一社会信用代码、注册地址、成立日期 |
| 注册资本 | 认缴资本、实缴资本、出资方式 |
| 经营范围 | 主营业务、经营许可 |
| 股权结构 | 股东信息、持股比例、股权变更历史 |
| 高管信息 | 法定代表人、董监高成员 |
| 变更记录 | 工商变更历史、备案信息 |

### 2. 股权穿透
- 股东层级穿透（至少3层）
- 实际控制人识别
- 一致行动人关系
- 关联企业网络
- 对外投资情况
- 分支机构信息

### 3. 法律诉讼
| 类型 | 调查内容 |
|------|---------|
| 民事诉讼 | 作为原告/被告的案件 |
| 刑事案件 | 涉及的刑事犯罪 |
| 行政诉讼 | 与政府机关的纠纷 |
| 仲裁案件 | 商事仲裁记录 |
| 执行案件 | 被执行人信息、执行标的 |
| 失信记录 | 失信被执行人名单 |
| 限制消费 | 限制高消费令 |

### 4. 行政监管
- 行政处罚记录
- 经营异常记录
- 严重违法记录
- 环保处罚
- 税务处罚
- 监管处分

### 5. 知识产权
- 专利信息（发明、实用新型、外观设计）
- 商标注册
- 著作权登记
- 软件著作权
- 域名备案

### 6. 财务状况
- 年度报告（如公开）
- 纳税信用等级
- 抵押质押登记
- 动产融资登记
- 土地房产信息

### 7. 舆情分析
- 新闻舆情监控
- 社交媒体评价
- 行业口碑
- 负面事件

## 风险评估体系

### 风险评分模型
```
总分 = 100 - Σ(风险扣分)

风险扣分规则：
- 诉讼案件（每起）：被告-3分，原告-1分
- 执行案件（每起）：-5分
- 失信记录（每条）：-10分
- 行政处罚（每次）：-5分
- 经营异常（每次）：-8分
- 负面舆情（每条）：-2分
```

### 风险等级
| 评分 | 等级 | 建议 |
|------|------|------|
| 90-100 | 🟢 低风险 | 可正常开展合作 |
| 70-89 | 🟡 中风险 | 需关注风险点，加强条款保护 |
| 50-69 | 🟠 较高风险 | 建议深入调查，谨慎决策 |
| 0-49 | 🔴 高风险 | 建议暂缓或终止合作 |

## 使用示例

### 示例1：投资尽调
```
用户: 我们计划投资XX科技公司，请进行尽职调查。

AI: 好的，我将对XX科技公司进行全面的投资尽职调查，主要关注以下方面：
1. 股权结构及实控人
2. 历史融资及估值
3. 法律诉讼风险
4. 核心资产（知识产权）
5. 财务状况

请问您是否有以下资料可以提供？
- 公司名称或统一社会信用代码
- 重点关注领域
- 调查深度要求
```

### 示例2：供应商调查
```
用户: 帮我调查一下这个新供应商的背景，看看是否可靠。

AI: 收到，我将对该供应商进行资质审查，重点关注：
1. 工商注册信息
2. 经营资质
3. 诉讼及信用记录
4. 行业口碑

请提供供应商的公司名称...
```

## 数据来源

本技能可集成以下数据源：
- 国家企业信用信息公示系统
- 中国裁判文书网
- 全国法院失信被执行人名单
- 国家知识产权局
- 天眼查/企查查API
- 新闻舆情数据库

## 注意事项

1. 调查信息仅供决策参考，不构成法律意见
2. 部分非公开信息可能需要授权获取
3. 信息存在时滞，建议核实最新状态
4. 重大决策建议结合专业机构意见
