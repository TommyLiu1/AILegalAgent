# 资产线索挖掘技能

## 技能概述

本技能用于辅助律师在诉讼保全或执行阶段，快速梳理被执行人的财产线索。它不直接连接银行系统（那是法院专网的事），而是通过公开数据（知识产权、对外投资、招投标）进行**逻辑推演**。

## 核心能力

### 1. 显性资产分析
- **股权资产**: 穿透分析被执行人对外持股情况（全资、控股、参股）。
- **知识产权**: 专利（发明、实用新型）、商标、软件著作权。提示这些资产的可变现价值。
- **不动产线索**: 通过注册地址、分支机构地址、招投标中标项目（工程类）推断不动产或工程款债权。

### 2. 隐形资产/债权推演
- **供应链债权**: 通过“中标公告”发现被执行人作为供应商，推断其对招标方（国企/央企）享有应收账款。
- **涉诉债权**: 查阅被执行人作为“原告”且胜诉的案件，推断其享有到期债权（可申请代位执行）。

## 挖掘规则 (Logic-Based)

### 应收账款线索
```
如果 被执行人 近1年内有中标记录:
    推断: "高概率持有应收账款"
    行动建议: "建议向招标方（通常为信誉较好的国企/事业单位）发送协助执行通知书，或申请冻结应收账款。"
```

### 知识产权价值
```
如果 被执行人 持有 "发明专利" 或 "驰名商标":
    推断: "具备无形资产处置价值"
    行动建议: "可申请查封其知识产权，并评估拍卖变现的可能性；或以此为谈判筹码逼迫其还款。"
```

### 关联公司资产
```
如果 被执行人 是某注册资本实缴公司的股东:
    推断: "持有股权资产"
    行动建议: "申请冻结其持有的该公司股权。"
```
