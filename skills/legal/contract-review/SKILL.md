# 合同审查技能

## 技能概述

本技能用于自动化审查各类商业合同文本，识别潜在法律风险条款，提取关键商业条款，并提供专业的修改建议。

## 适用范围

### 支持的合同类型
- 买卖合同
- 服务合同/协议
- 劳动合同
- 租赁合同
- 股权转让协议
- 投资协议
- 保密协议（NDA）
- 竞业禁止协议
- 知识产权许可协议
- 合作框架协议

### 支持的文件格式
- PDF (.pdf)
- Word (.doc, .docx)
- 纯文本 (.txt)
- Markdown (.md)

## 审查流程

### 1. 文档解析
- 解析合同文本结构
- 识别合同各组成部分
- 提取元数据（签约日期、合同编号等）

### 2. 条款识别与提取
提取以下关键条款：

| 条款类型 | 审查要点 |
|---------|---------|
| 主体条款 | 签约主体资格、授权代表 |
| 标的条款 | 标的物/服务内容、数量、规格 |
| 价款条款 | 金额、支付方式、支付时间 |
| 期限条款 | 合同期限、起止时间 |
| 履行条款 | 履行方式、地点、验收标准 |
| 违约条款 | 违约情形、违约金、赔偿责任 |
| 保密条款 | 保密义务、保密期限、例外情形 |
| 知识产权 | IP归属、使用权限、限制 |
| 不可抗力 | 定义、通知义务、后果 |
| 争议解决 | 仲裁/诉讼、管辖法院 |
| 解除条款 | 解除条件、解除程序、后果 |

### 3. 风险识别

#### 风险类别
1. **法律风险** - 违反强制性法律规定
2. **权益风险** - 条款明显不公平、对己方不利
3. **履行风险** - 条款模糊可能导致履行争议
4. **争议风险** - 争议解决条款不利或不明确

#### 风险等级
- 🔴 **高风险**: 可能导致重大损失或法律责任，需立即修改
- 🟡 **中风险**: 存在潜在问题，建议修改
- 🟢 **低风险**: 建议性优化，可选择修改

### 4. 生成建议
针对每个识别的风险点，生成：
- 风险描述
- 问题分析
- 修改建议（含具体条款措辞）
- 法律依据（如适用）

### 5. 输出报告
生成结构化审查报告，包含：
- 合同概述
- 关键条款摘要
- 风险清单（按等级排序）
- 修改建议
- 综合评估

## 审查规则

### 违约金审查规则
```
如果 违约金比例 > 合同金额的30%:
    标记为高风险
    建议: 降低违约金比例至20%-30%之间
    依据: 《民法典》第585条，违约金过高可被调整
```

### 管辖条款审查规则
```
如果 管辖法院为对方所在地:
    标记为中风险
    建议: 协商变更为原告所在地或合同履行地
```

### 保密期限审查规则
```
如果 保密期限 > 5年 且 无合理商业原因:
    标记为中风险
    建议: 考虑限定保密期限或明确保密信息范围
```

## 使用示例

### 示例1：服务合同审查
```
用户: 请审查这份IT服务合同，特别关注知识产权和保密条款。

AI: 收到，我将对该IT服务合同进行全面审查，并重点关注知识产权和保密条款...
```

### 示例2：劳动合同审查
```
用户: 帮我检查这份劳动合同是否合规。

AI: 好的，我将从以下几个方面审查该劳动合同：
1. 必备条款是否齐全
2. 试用期规定是否合法
3. 薪酬福利条款...
```

## 注意事项

1. 本技能生成的审查意见仅供参考，不构成正式法律意见
2. 对于重大合同，建议结合专业律师意见
3. 审查结果基于提供的合同文本，如有附件应一并提供
4. 不同行业可能有特殊监管要求，请注意行业特殊规定
